# Rust-Tauri-Sveltekit

> - 위 프로젝트는 sveltekit, Rust, Tauri 프로젝트입니다. by mingty

> - @sveltejs/kit 1.0.0
> - rust 1.65.0
> - Tauri

### 모듈 설치
``` sh
$ npm i
```

### 빌드
``` sh
$ npm run build
```

### 프로젝트 실행
``` sh
$ npm start
```

### Rust 프로젝트 실행
``` sh
$ cargo run
```

### Rust 프로젝트 build (window, *: default)
```sh
$ cargo build
```

### Rust 프로젝트 build (Linux)
```sh
$ cargo build --target x86_64-unknown-linux-gnu
```
